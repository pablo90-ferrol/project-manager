mostrarTarea(req, res)

Controlador para mostrar la información de una tarea en concreto. La información de la tarea se obtiene mediante una consulta a la base de datos, con todas las intervenciones asociadas a dicha tarea. La consulta trabaja de forma asincrona, de tal modo que los datosde la tarea se obtienen en una persona. Los datos obtenidos de la consulta se visualizan en la vista.
Parameters:
Name Type Description
req * Petición, que contiene el parámetro del ID de la tarea.
res * Respuesta.
